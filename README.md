# The Demon Pill and the Spirit Pearl
四川大学大一程序设计期末游戏项目

项目名称：魔丸与灵珠——旋转吧我的球

项目简介：该项目采用了C语言的和第三方的EasyX图形库，实现了两个小球旋转躲避障碍物。

小游戏的功能：
1. 小球顺时针，逆时针左右键控制
2. 小球s键加速旋转
3. 游戏的得分记录，最高分显示，速度显示
4. 每局游戏结束用户自己选择继续还是退出
